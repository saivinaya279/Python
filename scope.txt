1.local Scope
"""Programming definition (for notes)

A variable created inside a function is called a local variable 
and can only be accessed inside that function."""
def fun():

    x = 10   # local variable

    print(x)

fun()
Important rule

Local variable cannot be accessed outside.
//Why?

Because x exists only inside function.

Memory line:

Local = Inside function only
//2️⃣ Global Scope
A variable declared outside all functions is called 
a global variable and can be accessed anywhere in the program.
x = 50   # global

def fun():

    print(x)
